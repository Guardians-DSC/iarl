O que é a IARL?

O IARL (Interface de Acesso Remoto a Laboratórios) é uma aplicação web que fornece uma interface amigável de acesso a arquivos e diretórios aos usuários dos LCCs (Laboratórios de Ciência da Computação). A célula do IARL é uma unidade interna do Guardians que foca no desenvolvimento do projeto IARL. Nesse projeto, os integrantes da célula desenvolvem conhecimentos de git (e Github), servidores em NodeJS e frontends usando frameworks JavaScript (Vue, React ou Angular)
